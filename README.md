# JavaScript Hands-On

A collection of hands-on HTML & JavaScript challenges designed to strengthen
fundamentals such as DOM manipulation, events, and conditional logic.

## Goals
- Practice vanilla JavaScript
- Improve problem-solving and logic
- Build confidence before moving to frameworks

## Challenges
1. Click Counter
2. Show / Hide Password
3. Grade Evaluator
4. Dynamic To-Do List
5. Character Counter

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
